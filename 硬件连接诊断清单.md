# DMT143 硬件连接诊断清单

## 🔴 当前问题

**现象**: 串口 COM3 可以打开,但设备无任何响应

**测试结果**:
- ✅ 串口打开成功
- ❌ I 命令无响应(设备信息)
- ❌ FORM 命令无响应(格式查询)
- ❌ SEND 命令无响应(单次读取)
- ❌ R 命令无响应(连续输出)
- ❌ 无自发数据

---

## ⚠️ 可能原因

### 1. 硬件连接问题 (最可能)

#### 检查项:
- [ ] **DMT143 是否通电?**
  - 查看设备上的指示灯
  - 确认电源线连接
  - 确认电源适配器工作正常

- [ ] **串口线是否连接正确?**
  - USB转串口线是否插好
  - COM3 是否对应正确的物理接口
  - 尝试重新插拔USB

- [ ] **串口线类型是否正确?**
  - DMT143 使用 RS232 还是 RS485?
  - 是否需要交叉线(TX-RX)?
  - 是否需要转换器?

- [ ] **接线顺序是否正确?**
  ```
  标准RS232接线:
  DMT143 TX  → 串口 RX
  DMT143 RX  → 串口 TX
  DMT143 GND → 串口 GND
  ```

### 2. 波特率不匹配

#### 测试建议:
运行 `hardware_diagnose.py`,它会自动测试:
- 9600 baud
- 19200 baud (默认)
- 38400 baud
- 57600 baud
- 115200 baud

如果都无响应,可能是:
- 非标准波特率
- 设备波特率被修改过
- 需要查看设备手册确认

### 3. 设备模式问题

#### DMT143 可能处于特殊模式:

**Modbus模式**:
- 设备可能配置为Modbus RTU模式
- 需要Modbus协议通信
- 查看设备手册如何切换到ASCII模式

**停止模式**:
- 设备可能处于休眠或停止状态
- 可能需要特定的唤醒命令
- 尝试断电重启

**配置模式**:
- 设备可能被锁定在配置模式
- 查看是否有硬件跳线或开关
- 参考设备手册

### 4. 串口驱动问题

#### 检查:
- [ ] 设备管理器中确认 COM3 状态
  ```
  打开: 设备管理器 → 端口(COM和LPT)
  查看: XR21B1411 USB UART (COM3)
  状态: 应该显示"此设备工作正常"
  ```

- [ ] 尝试重新安装驱动
  - 从官网下载最新驱动
  - 卸载当前驱动后重新安装

- [ ] 尝试其他USB口
  - 换一个USB端口
  - 避免使用USB集线器

### 5. 设备故障

#### 测试:
- [ ] 使用其他串口调试工具
  - **Putty**: 免费串口终端
  - **TeraTerm**: 功能强大的终端软件
  - **串口调试助手**: 国产工具

- [ ] 尝试其他电脑
  - 排除电脑问题
  - 排除驱动问题

- [ ] 联系技术支持
  - 设备可能损坏
  - 需要专业维修

## 🔧 立即行动

### 步骤1: 运行硬件诊断工具
```bash
cd e:/下载/app/DMT143_Monitor_v1.0
python hardware_diagnose.py
```

选择 COM3,工具会:
- 测试多种波特率
- 发送多种命令
- 监听自发数据
- 提供诊断建议

### 步骤2: 检查硬件

**查看设备**:
1. DMT143 设备上的指示灯状态
   - 电源指示灯是否亮?
   - 通信指示灯是否闪烁?
   - 有无错误指示?

2. 串口线连接
   - USB转串口线型号
   - 连接到DMT143的接口类型
   - 是否有转换器?

**拍照记录**:
- 设备指示灯状态
- 接线情况
- 设备型号标签

### 步骤3: 使用其他工具测试

#### Putty 测试步骤:
1. 下载 Putty: https://www.putty.org/
2. 打开 Putty
3. 选择 "Serial"
4. 设置:
   ```
   Serial line: COM3
   Speed: 19200
   Data bits: 8
   Stop bits: 1
   Parity: None
   Flow control: None
   ```
5. 点击 "Open"
6. 输入命令: `I` 然后回车
7. 观察是否有响应

如果 Putty 也无响应 → 确认是硬件问题

### 步骤4: 查看设备手册

检查以下内容:
- [ ] 默认波特率是多少?
- [ ] 使用 RS232 还是 RS485?
- [ ] 是否需要特殊的初始化命令?
- [ ] 如何恢复出厂设置?
- [ ] 指示灯状态含义?
