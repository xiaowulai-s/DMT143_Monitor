## 🔍 问题分析

根据之前的测试结果:
- ✅ 串口可以打开

- ❌ 无自发数据

**可能原因**: DMT143 设备处于 **Modbus RTU 模式**,而不是 ASCII 模式。

---

## 📚 DMT143 通信模式说明

Vaisala DMT143 支持多种通信模式:

### 1. ASCII 模式 (推荐)
- **特点**: 人类可读的文本命令
- **命令**: `I`, `R`, `S`, `FORM`, 等
- **示例**: `Tdf= 6.26 'C Tdfatm= 6.45 'C H2O= 9611 ppm`
- **优点**: 易于调试和使用
- **默认波特率**: 19200

### 2. Modbus RTU 模式
- **特点**: 二进制协议,需要计算CRC
- **命令**: 功能码 (03, 04, 06, 等)
- **示例**: `01 03 00 00 00 0A C4 CD` (读10个寄存器)
- **优点**: 工业标准,可靠性高
- **默认从机地址**: 通常为 1 或 2

### 3. Modbus ASCII 模式
- **特点**: ASCII编码的Modbus协议
- **格式**: 以 `:` 开头,以 `\r\n` 结尾

### 方法2: 手动切换

如果自动工具无法切换,尝试以下方法:

#### 步骤1: 断电重启
```
1. 断开 DMT143 电源
2. 等待 10 秒
3. 重新上电
4. 快速运行: python modbus_switch.py
```

#### 步骤2: 短接跳线
某些设备有硬件跳线:
```
1. 查看设备电路板上的跳线
2. 找到标有 "MODE" 或 "COMM" 的跳线
3. 短接或断开跳线
4. 重启设备
```

#### 步骤3: 使用配置软件
Vaisala可能提供配置软件:
```
1. 从Vaisala官网下载配置工具
2. 连接设备
3. 在软件中切换通信模式
```

#### 步骤4: 通过Modbus写寄存器
如果设备在Modbus模式,可以通过写寄存器切换:

**常用寄存器地址** (因设备型号而异):
```
寄存器 100: 模式选择
  - 值 0: ASCII模式
  - 值 1: Modbus RTU模式
  - 值 2: Modbus ASCII模式

寄存器 200: 通信协议
  - 值 0: 默认协议
  - 值 1: Modbus

寄存器 1000: 系统配置
```

**使用Modbus Poll工具**:
1. 下载 Modbus Poll (modbus工具)
2. 连接 COM3,波特率 19200
3. 从机地址设为 1
4. 功能码: 06 (写单个寄存器)
5. 写入寄存器地址和值
6. 观察设备响应

## 🛠️ 故障排查

### 问题1: 未检测到Modbus模式

**可能原因**:
- 设备不在Modbus模式
- 波特率不匹配
- 从机地址不在测试范围
- 硬件连接问题

**解决方案**:
```bash
# 测试其他波特率
python hardware_diagnose.py

# 检查硬件连接
- 确认设备通电
- 确认串口线正确
- 尝试其他串口
```

### 问题2: 检测到Modbus但无法切换

**可能原因**:
- 设备锁定在Modbus模式
- 需要特定密码或权限
- 需要硬件跳线

**解决方案**:
```
1. 断电重启设备
2. 查看设备手册
3. 联系Vaisala技术支持
4. 使用Vaisala配置软件
```

### 问题3: 切换后仍无法使用ASCII命令

**可能原因**:
- 切换未生效
- 波特率改变
- 需要保存配置

**解决方案**:
```bash
# 测试所有波特率
python modbus_switch.py

# 会自动测试: 19200, 9600, 38400
```

## 📖 Modbus协议基础知识

### Modbus RTU 帧格式
```
[从机地址] [功能码] [数据] [CRC16]
   1字节     1字节    N字节   2字节
```

### 常用功能码
| 功能码 | 名称 | 说明 |
|--------|------|------|
| 03 | 读保持寄存器 | 读取设备参数 |
| 04 | 读输入寄存器 | 读取测量数据 |
| 06 | 写单个寄存器 | 设置参数 |
| 16 | 写多个寄存器 | 批量设置 |

### DMT143 寄存器映射 (示例)
```
输入寄存器 (只读):
  30001: 露点温度 Tdf
  30002: 标准气压露点 Tdfatm
  30003: 体积含水量 H2O
  30004: 设备状态

保持寄存器 (可读写):
  40001: 设备地址
  40002: 波特率
  40003: 通信模式
  40004: 输出格式
```

**注意**: 实际寄存器地址需查阅设备手册!

## 🎯 快速参考

### 检测Modbus模式
```python
# Modbus RTU 命令示例
命令: 01 03 00 00 00 0A C4 CD
说明: 从机地址01, 读保持寄存器, 起始地址0, 读10个
```

### 切换到ASCII模式
```python
# 方法1: ASCII命令
发送: EXIT\r 或 MODE ASCII\r

# 方法2: Modbus写寄存器
命令: 01 06 00 64 00 00 88 1E
说明: 从机01, 写寄存器, 地址100(0x64), 值0
```

### 测试ASCII模式

```python
# 基本命令
I     - 查询设备信息
?     - 显示帮助
FORM  - 查询格式
R     - 开始输出
S     - 停止输出
```
