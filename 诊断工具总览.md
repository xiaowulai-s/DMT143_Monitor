## 🔧 Modbus切换核心命令

如果需要手动操作,核心命令如下:

### 检测 Modbus RTU
```
命令(十六进制): 01 03 00 00 00 0A C4 CD
说明: 读从机01的10个保持寄存器
```

### 切换到 ASCII
```
方法1: 发送 ASCII 命令
  EXIT\r
  MODE ASCII\r

方法2: Modbus 写寄存器
  命令: 01 06 00 64 00 00 88 1E
  说明: 写寄存器100=0 (地址和值因设备而异)
```
