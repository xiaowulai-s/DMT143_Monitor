## 功能特性

### 数据采集与显示
- **实时仪表盘**：三个仪表盘同时显示 Tdf（露点温度）、Tdfatm（标准气压露点）、H2O（体积含水量）
- **动态曲线图**：实时绘制露点温度变化趋势，类似心电图形式
- **迷你趋势图**：仪表盘下方显示小范围数据波动

### 连接与通信
- **自动扫描串口**：启动时自动列出可用串口
- **RS485/RS232 支持**：自动适配工业级 RS485 半双工通信
- **自动重连**：设备断开后自动检测并重新连接

### 日志管理
- **实时日志**：底部滚动显示原始数据帧和解析后数据
- **历史日志查看器**：菜单 → 文件 → 历史日志
- **多选操作**：支持 Ctrl/Shift 多选，单选/多选删除，全选/取消

### 用户界面
- **浅蓝色专业主题**：视觉舒适的工业风格配色
- **自定义图标**：应用窗口显示自定义图标
- **品牌标识**：菜单栏右侧显示 "Power By QianYiHui"
- **设备信息面板**：显示型号、序列号、波特率、地址、输出间隔

## 使用流程

```
┌─────────────────────────────────────────────────────────────┐
│                      1. 启动程序                            │
│                   双击 exe 或 python main.py               │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      2. 选择串口                            │
│           从下拉框选择 DMT143 设备对应的 COM 口             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      3. 连接设备                            │
│                    点击绿色"连接"按钮                       │
│         连接成功后状态变为绿色，显示设备信息                 │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      4. 监控数据                            │
│           仪表盘实时显示数值，曲线图绘制趋势                 │
│              底部日志实时滚动数据帧                          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      5. 断开/退出                           │
│            点击红色"断开"按钮，或 Ctrl+Q 退出               │
│            断开时日志自动保存到 logs 文件夹                  │
└─────────────────────────────────────────────────────────────┘
```

## 快捷键

- Ctrl+L: 查看历史日志
- Ctrl+R: 刷新串口
- Ctrl+Q: 退出程序
