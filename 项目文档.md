# DMT143 露点监控系统

## 项目介绍

DMT143 露点监控系统是一款用于连接和监控 DMT143 露点变送器的软件工具。该系统可以实时采集设备数据，展示测量结果，并支持数据记录和导出功能。

DMT143 是芬兰维萨拉（Vaisala）公司生产的露点变送器，广泛应用于工业领域的气体湿度监测。本系统提供了友好的图形界面，方便用户快速查看设备状态和数据变化。

---

## 功能说明

### 1. 实时数据监控

系统可以实时显示以下三个测量参数：

- **露点温度（Tdf）**：气体冷却至饱和时的温度，单位 °C
- **标准气压露点（Tdfatm）**：标准大气压下的露点温度，单位 °C
- **体积含水量（H2O）**：气体中的水蒸气含量，单位 ppm

### 2. 可视化展示

- **仪表盘显示**：三个独立的仪表盘，分别显示三个测量值，直观易懂
- **实时曲线**：类似心电图的形式，实时展示露点温度的变化趋势
- **迷你趋势图**：在仪表盘下方显示小范围的数据波动

### 3. 系统日志

- 实时显示原始数据帧
- 解析后的数据（带单位）
- 连接状态和错误信息

### 4. 报警功能

- 用户可以设置报警阈值（默认范围：-80°C 到 20°C）
- 当测量值超出设定范围时，对应的数值会变成红色并显示警告提示
- 报警功能默认关闭，可在设置中开启

### 5. 数据管理

- **数据导出**：支持将历史数据导出为 CSV 格式，可在 Excel 中打开
- **自动保存**：系统会自动保存用户的设置（报警阈值、刷新频率等），下次打开无需重新配置

### 6. 连接控制

- 自动扫描并显示可用的串口
- 支持手动刷新串口列表
- 支持连接和断开操作
- 断线后会自动尝试重连

## 使用指南

### 1. 准备工作

确保 DMT143 设备已通过串口线连接到电脑，并确认串口号。

### 2. 启动程序

```bash
cd e:/下载/app/DMT143_Monitor_v1.0
python main.py
```

或直接双击运行 `main.py`

### 3. 连接设备

1. 在"串口"下拉菜单中选择设备对应的串口号（通常为 COM3）
2. 点击"▶ 连接"按钮
3. 连接成功后，状态指示灯会变成绿色

### 4. 查看数据

连接成功后，系统会自动开始读取数据：

- 仪表盘会显示当前的测量值
- 曲线图会开始绘制数据变化
- 底部会显示实时的露点温度
- 系统日志会显示解析后的数据

### 5. 配置报警（如需要）

1. 点击菜单栏的"设置" → "报警设置"
2. 勾选"启用报警"
3. 设置报警的上限和下限
4. 点击"确定"保存

### 6. 导出数据

1. 点击菜单栏的"文件" → "导出数据"
2. 选择保存位置和文件名
3. 点击保存，数据将导出为 CSV 格式

## DMT143 通信说明

### 设备参数

| 参数 | 值 |
|------|-----|
| 波特率 | 19200 |
| 数据位 | 8 |
| 校验位 | N |
| 停止位 | 1 |
| 通信模式 | RS485 半双工 |

### 数据格式

设备输出格式：`Tdf Tdfatm H2O`

示例：`21.10 21.30 25626.0`

- Tdf：露点温度 (°C)
- Tdfatm：标准气压露点 (°C)
- H2O：体积含水量 (ppm)

### 命令序列

连接后系统自动执行：

1. `FORM TDF TDFA H2O` - 设置输出格式
2. `R` - 开始连续输出

详细命令说明请参考 `commands.md`。

## 常见问题

### Q: 连接不上设备怎么办？

A: 请检查以下几点：
1. 确认串口线连接正常
2. 确认设备已通电
3. 检查设备是否被其他程序占用
4. 尝试刷新串口列表，重新选择
5. 检查波特率设置（默认19200）

### Q: 数据曲线不显示怎么办？

A: 确保已成功连接设备并且设备正在输出数据。如果连接正常但仍不显示，可以尝试重新连接。

### Q: 报警一直响怎么办？

A: 进入"设置" → "报警设置"，取消勾选"启用报警"即可关闭报警功能。

### Q: 数据解析错误怎么办？

A:
1. 检查设备是否正确输出数据格式
2. 尝试重新连接设备
3. 参考commands.md确认命令格式正确
